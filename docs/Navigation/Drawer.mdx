The Drawer component displays a panel sliding in from the side or bottom, often containing actions or navigation items.

## When to Use

- Show additional information or actions without leaving the current page
- Use for navigation menus, filter options, or settings panels
- Temporary content that should not disrupt the main content area

## ðŸ“– Overview

`Drawer` is a panel that overlays the content area and slides in from the edge of the screen. It is commonly used for navigation, filtering, or displaying secondary information. Unlike a Modal, a Drawer typically occupies one side of the screen while keeping the background content partially visible.

### ðŸ“Œ Design Principles

* **Accessibility:** Must support keyboard focus management and WAI-ARIA best practices.
* **Flexibility:** Supports sliding from left, right, top, and bottom.

| `onClose` | `() => void` | `() => {}` | Callback fired when the drawer closes. | âœ… Yes |
| `size` | `number \| string` | `'300px'` | Width (for horizontal) or height (for vertical) of the drawer. | âŒ No |
| `children` | `ReactNode` | N/A | Content to render inside the drawer. | âœ… Yes |

> **Note:** When `direction` is `'left'` or `'right'`, `size` controls width; when `'top'` or `'bottom'`, `size` controls height.

### 2. Methods

| Method | Params | Returns | Description |
| :--- | :--- | :--- | :--- |
| `open()` | none | `void` | Imperatively opens the drawer. *(only available in uncontrolled mode)* |
| `close()` | none | `void` | Imperatively closes the drawer. *(only available in uncontrolled mode)* |

## ðŸ’¡ Usage Examples

Using the Drawer is straightforward: control its `isOpen` state and provide `onClose`.

### 1. Basic Usage

This example shows a drawer sliding in from the right:

```jsx

  const [open, setOpen] = useState(false);

  return (
    <>
      <button onClick={() => setOpen(true)}>Open right drawer</button>

      <Drawer
        isOpen={open}
        onClose={() => setOpen(false)}
        direction="right"
        size="40vw"
      >
        <h1>ðŸ‘‹ Welcome to Drawer!</h1>
        <p>This is the main content inside the drawer.</p>
        <ul>
          <li>List item A</li>
          <li>List item B</li>
        </ul>
        <button onClick={() => setOpen(false)}>Close</button>
      </Drawer>
    </>
  );
}
```
