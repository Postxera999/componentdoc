# Drawer (Blueprint)

基于 Figma 节点 `308:55` 的 Drawer 原型样式复刻，布局、尺寸、间距、字体与颜色严格对齐 Figma：
- 尺寸：宽 `375px`，顶部圆角 `16px 16px 0 0`
- 布局：容器竖排，`gap: 8px`，`padding: 16px 0 40px`
- 标题：`Fira Sans 20/Bold`，文字颜色 `#000000`
- 列表项：`Fira Sans 18/Regular`，选中态背景 `#F3F3F3`
- 图标：尺寸 `24×24`，位置左侧（使用 Plus 图标拟合 Figma Icon/Add）

## 说明

- 字体使用 `Fira Sans`，已通过 CSS 引入；如需线下环境完全一致，请确保字体可用。
- 图标使用 `lucide-react` 的 `Plus` 近似 Figma 的 `Icon/Add`；若需像素级一致，可替换为 Figma 导出的 SVG。
- 该组件为演示静态结构（Blueprint），不包含交互行为（开关、动画、遮罩点击关闭等）。如需完整交互版本，可在此基础上扩展。
