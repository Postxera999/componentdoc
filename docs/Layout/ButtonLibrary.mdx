A complete button component library based on the Figma UI Components Library, offering multiple variants and states for standard buttons and icon buttons.

## Preview

<Demo>
  <ButtonLibrary />
</Demo>

## Features

- Pixel-perfect parity with the Figma design
- Multiple variants: Primary, Secondary, Tertiary
- Interaction states: Default, Active, Disabled
- Interactive: click to toggle the Active state
- Icon support: native SVG icons exported from Figma
- Responsive design for mobile and desktop
- Accessibility: keyboard navigation and screen reader support

## Component Structure

### Standard Button

Standard buttons provide two main variants:

- **Primary** — main action button with solid background
- **Secondary** — secondary action button with outlined style

Each variant supports three states:
- **Default** — default/resting state
- **Active** — active/pressed state (toggle by clicking)
- **Disabled** — disabled, not interactive

### Icon Button

Icon buttons are designed for single-icon actions and support three variants:

- **Primary** — solid background icon button
- **Secondary** — secondary icon button on dark background
- **Tertiary** — outlined tertiary icon button

## Usage

### Basic

```tsx
import ButtonLibrary from '@/content/demos/Layout/ButtonLibrary'

// Complete button library showcase
<ButtonLibrary />
```

### Use buttons individually

```tsx
// Standard button
<Button variant="primary" state="default">
  Button Text
</Button>

// Button with icon
<Button variant="primary" state="default" hasIcon>
  Button Text
</Button>

// Icon button
<IconButton variant="primary" state="default" />
```

### Interactive button

```tsx
const [isActive, setIsActive] = useState(false)

<Button 
  variant="primary" 
  state={isActive ? 'active' : 'default'}
  onClick={() => setIsActive(!isActive)}
>
  Toggle Button
</Button>
```

## API Reference

### Button Props

| Property | Type | Default | Description |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary'` | `'primary'` | Button variant |
| `state` | `'default' \| 'active' \| 'disabled'` | `'default'` | Button state |
| `hasIcon` | `boolean` | `false` | Whether to show an icon |
| `children` | `React.ReactNode` | - | Button text content |
| `onClick` | `() => void` | - | Click handler |
| `className` | `string` | `''` | Custom CSS class name |

### IconButton Props

| Property | Type | Default | Description |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary' \| 'tertiary'` | `'primary'` | Icon button variant |
| `state` | `'default' \| 'active' \| 'disabled'` | `'default'` | Button state |
| `onClick` | `() => void` | - | Click handler |
| `className` | `string` | `''` | Custom CSS class name |

## Design Guidelines

### Color System

- **Primary**: `#131414`
- **Primary-95**: `#131414`
- **Primary-80**: `#424343` (text)
- **Primary-50**: `#808080` (border)
- **Primary-05**: `#F3F3F3` (subtle backgrounds)
- **Background**: `#FFFFFF`

### Typography

- **Family**: Fira Sans
- **Weights**: 400 (Regular), 500 (Medium)
- **Sizes**: 12px (label), 14px (button text)
- **Line height**: 1.2

### Sizing

- **Standard button**: min-height 36px, horizontal padding 16px
- **Icon button**: 48x48px circular
- **Radius**: 25px (standard), 100px (icon button)
- **Spacing scale**: 8, 12, 16, 24

### Interaction

- **Hover**: subtle 1px lift + color change
- **Focus**: 2px outer focus ring
- **Transition**: 0.2s ease-in-out
- **Disabled**: 25% opacity

## Accessibility

- Keyboard Tab navigation
- Visible focus indicator
- Correct `disabled` attribute for disabled state
- Icon buttons include appropriate `aria-label`
- High-contrast mode support
- Reduced motion preference support

## Browser Compatibility

- Chrome 88+
- Firefox 85+
- Safari 14+
- Edge 88+
