# Button Library

基于 Figma UI Components Library 设计的完整按钮组件库，提供了标准按钮和图标按钮的多种变体和状态。

## 预览

<Demo>
  <ButtonLibrary />
</Demo>

## 特性

- **完全基于 Figma 设计**：像素级还原 Figma 设计稿的视觉效果
- **多种变体**：支持 Primary、Secondary、Tertiary 等变体
- **交互状态**：包含 Default、Active、Disabled 状态
- **可交互**：点击按钮可切换 Active 状态
- **图标支持**：使用 Figma 导出的原生 SVG 图标
- **响应式设计**：适配移动端和桌面端
- **无障碍支持**：支持键盘导航和屏幕阅读器

## 组件结构

### 标准按钮 (Button)

标准按钮支持两种主要变体：

- **Primary**：主要操作按钮，使用深色背景
- **Secondary**：次要操作按钮，使用边框样式

每种变体都包含三种状态：
- **Default**：默认状态
- **Active**：激活状态（点击切换）
- **Disabled**：禁用状态

### 图标按钮 (IconButton)

图标按钮专为单一图标操作设计，支持三种变体：

- **Primary**：深色背景的主要图标按钮
- **Secondary**：深色背景的次要图标按钮  
- **Tertiary**：边框样式的第三级图标按钮

## 使用方法

### 基础用法

```tsx
import ButtonLibrary from '@/content/demos/Layout/ButtonLibrary'

// 完整的按钮库演示
<ButtonLibrary />
```

### 单独使用按钮组件

```tsx
// 标准按钮
<Button variant="primary" state="default">
  Button Text
</Button>

// 带图标的按钮
<Button variant="primary" state="default" hasIcon>
  Button Text
</Button>

// 图标按钮
<IconButton variant="primary" state="default" />
```

### 交互式按钮

```tsx
const [isActive, setIsActive] = useState(false)

<Button 
  variant="primary" 
  state={isActive ? 'active' : 'default'}
  onClick={() => setIsActive(!isActive)}
>
  Toggle Button
</Button>
```

## API 参考

### Button Props

| 属性 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary'` | `'primary'` | 按钮变体 |
| `state` | `'default' \| 'active' \| 'disabled'` | `'default'` | 按钮状态 |
| `hasIcon` | `boolean` | `false` | 是否显示图标 |
| `children` | `React.ReactNode` | - | 按钮文本内容 |
| `onClick` | `() => void` | - | 点击事件处理函数 |
| `className` | `string` | `''` | 自定义 CSS 类名 |

### IconButton Props

| 属性 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary' \| 'tertiary'` | `'primary'` | 图标按钮变体 |
| `state` | `'default' \| 'active' \| 'disabled'` | `'default'` | 按钮状态 |
| `onClick` | `() => void` | - | 点击事件处理函数 |
| `className` | `string` | `''` | 自定义 CSS 类名 |

## 设计规范

### 颜色系统

- **Primary**: `#131414` (深灰色)
- **Primary-95**: `#131414` (主色调)
- **Primary-80**: `#424343` (文本色)
- **Primary-50**: `#808080` (边框色)
- **Primary-05**: `#F3F3F3` (浅背景色)
- **Background**: `#FFFFFF` (背景白色)

### 字体规范

- **字体家族**: Fira Sans
- **字重**: 400 (Regular), 500 (Medium)
- **字号**: 12px (标签), 14px (按钮文本)
- **行高**: 1.2

### 尺寸规范

- **标准按钮**: 最小高度 36px，水平内边距 16px
- **图标按钮**: 48x48px 圆形
- **圆角**: 25px (标准按钮), 100px (图标按钮)
- **间距**: 8px, 12px, 16px, 24px

### 交互效果

- **悬停**: 轻微上移 1px + 颜色变化
- **焦点**: 2px 外边框高亮
- **过渡**: 0.2s ease-in-out
- **禁用**: 25% 透明度

## 无障碍特性

- 支持键盘 Tab 导航
- 提供焦点可见指示器
- 禁用状态正确设置 `disabled` 属性
- 图标按钮包含适当的 `aria-label`
- 支持高对比度模式
- 支持减少动画偏好设置
